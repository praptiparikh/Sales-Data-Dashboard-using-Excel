# Sales-Data-Dashboard-using-Excel
This project presents an interactive Sales Data Dashboard built using Excel to analyze key business performance metrics. The dashboard provides a comprehensive view of sales trends, revenue distribution, top-performing products, customer segments, and regional performance. It enables businesses to monitor growth, identify patterns, and make data-driven decisions.

Key Features

Interactive Dashboard with slicers and dynamic charts

KPIs Summary: Total Sales, Revenue, Profit, Orders, and Growth

Trend Analysis: Monthly/Quarterly sales performance

Product Insights: Best-selling products and categories

Regional Analysis: Sales comparison across regions

Customer Insights: Purchase patterns and contribution to revenue

Clean and professional visualization for decision-making
